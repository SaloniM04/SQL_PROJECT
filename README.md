# Restaurant Revenue Prediction using SQL 
OBJECTIVE- In this project, I aim to predict the restaurant revenue based on the independent features using SQL Server.

Columns include:- 
- Id            int
- Name          Varchar(100)
- Franchise     int
- Franchise2    varchar(30)
- Category      varchar(30)
- City          varchar(30)
- No_Of_Item    int
- Order_Placed  int
- Revenue       int
